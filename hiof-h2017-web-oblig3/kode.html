Kode

For å få god SEO er det veldig viktig å bruke riktige HTML tags og riktig struktur. Korrekt struktur betyr meningsfull og semantisk kode. Mange gjør feil og overbruker “div” tag. Eller bruker “br” tag for mellomrom,begge straffer nettstedet ditt i rangering på Google.

Det er viktig å bruke "article" tag for artikler, "nav" tag for navigasjon, "footer" tag for bunntekst og så videre. Små ting som "alt =" i "img" tag kan hjelpe mye, å ha noen søkeord om bilde hjelper søkemotoren å forstå hva det er. Dette resulterer i mer informasjon for søkemotorer om nettstedet ditt, og det er hele poenget med SEO. Jo mer informasjon søkemotoren har om nettstedet ditt, desto mer nøyaktige resultater kan de gi i søkene sine.

Her er min struktur for index for dette prosjektet, jeg bruker Gist i stedet for "code" tag, siden Gist syntaks er så mye bedre.
