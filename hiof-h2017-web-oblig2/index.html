Om denne siden Les mer

I denne artikkelen snakker jeg om dette nettstedets design og hvordan jeg kom opp med det. For å oppsummere, designet er basert på Material Design fra Google som brukes overalt av Google. Tanken er å gjenskape papir utseende på et nettsted med minimalistisk design.

Hva er MathML? Les mer

Denne artikkelen handler om MathML, og hva det er, jeg går i dybden på hvorfor det er nyttig i bruk sammen med HTML. På slutten av artikkelen gir jeg noen få eksempler av MathML.

Hva er SVG? Les mer

Denne artikkelen handler om SVG. Jeg forklarer hva SVG er og hvorfor det er en stort for webdesign. SVG er bare en type grafikk som JPG, PNG, GIF og så videre. Den store forskjellen er SVG kan manipuleres av CSS og JavaScript. På slutten av artikkelen viser jeg få eksempler, inkludert en fra personlig prosjekt.

Hva er Git og GitHub? Les mer

Git er stort for programmerere, Git ble laget av samme person som også startet Linux. Mange tror Git og GitHub er det samme. Det er galt, GitHub er basert på Git teknologi. Jeg forklarer mer detaljert på hva Git og GitHub er i artikkelen.
