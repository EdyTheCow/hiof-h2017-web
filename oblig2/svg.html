Hva er SVG?

SVG står for «Scalable Vector Graphics» som er basert på XML, og brukes til å vise grafikk på nettet. SVG dokumenter er bare enkle tekstfiler som beskriver linjer, kurver, former, farger og tekst. Det er også menneskelig lesbar, SVG kode kan bli manipulert via CSS eller JavaScript. Dette gir SVG en fleksibilitet som aldri kan matches av tradisjonelle PNG, GIF eller JPG.

SVG bruker former, tall og koordinater i stedet for piksel til å lage grafikk i nettleseren. Noe som gjør det uendelig scaleable uten å miste kvalitet. Med SVG kan du kombinere ulike former, baner og tekstelementer for å lage alle slags grafikk og være sikker på at de vil se klart og skarpt ut i hvilken som helst størrelse.
