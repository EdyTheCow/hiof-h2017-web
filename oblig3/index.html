Kom i gang Les mer

SEO står for "Search Engine Optimization". Det er et sett med regler som hjelper deg å rangere høyere i søkeresultatene. Disse regler inkluderer ting som god kodestruktur, riktig bruk av HTML-tagger. Det ser også på søkeord, domene og til og med URL.

Kode Les mer

For å få god SEO er det veldig viktig å bruke riktige HTML tags og riktig struktur. Korrekt struktur betyr meningsfull og semantisk kode. Mange gjør feil og overbruker “div” tag. Eller bruker “br” tag for mellomrom,begge straffer nettstedet ditt i rangering på Google.

Innhold Les mer

Innholdet på nettstedet ditt er like viktig som kodestrukturen din. Det er ikke så mange som tenker på det, men dine titles, subtitles og lister på nettstedet ditt er viktigere enn du tror. Når søkemotorer som Google gjennomsøker nettstedet ditt. Den ser på dine titles, subtitles og lister for søkeord.

Analyse Les mer

Først og fremst fokuserte jeg på god kodestruktur som er lett å forstå og lese. Jeg har også brukt semantiske tags for å best representere alle elementene på nettstedet. God struktur og semantisk tags gir bedre SEO.
